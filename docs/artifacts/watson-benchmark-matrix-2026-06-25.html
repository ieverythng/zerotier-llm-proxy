Scoring formula

Score = 0.25 * normalized_decode_tok_s + 0.20 * normalized_prefill_tok_s + 0.20 * tool_success_rate + 0.15 * context_success_score + 0.10 * stability_score + 0.10 * VRAM_safety_score Promotion rule: - total score > baseline + 8% - no critical regression in tool success - no OOM/restart in 3 repeated runs - rollback script verified

Run order

1
Run B0 three times and save JSON metrics.
2
Run C1/C2 before any Q2 experiment. Context first, quantization second.
3
Run Headroom on/off for identical Discord tasks and compare total prompt tokens.
4
Run Q2 only after the Q3 context ladder fails or leaves insufficient VRAM.
5
Run vLLM sidecar and DFlash in isolation; never over the live Discord route until they pass.
